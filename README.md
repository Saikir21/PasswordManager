# Local Password Manager (Python / SQLite / Fernet)

Локальный менеджер паролей с консольным интерфейсом (CLI), построенный на принципах безопасного хранения данных и разделения архитектурных слоев.

## Архитектура проекта
* `main.py` — Управляющий консольный интерфейс (CLI) и логика сессии.
* `core.py` — Ядро приложения: криптография (Fernet, PBKDF2), взаимодействие с SQLite.
* `models.py` — Описание сущностей (Data Classes) для передачи данных.
* `config.py` — Централизованные настройки безопасности и константы.

## Функции безопасности
* **Симметричное шифрование:** Данные шифруются алгоритмом AES (через `cryptography.fernet`).
* **Усиление ключа:** Мастер-ключ генерируется через `PBKDF2HMAC` с использованием SHA-512 и 600 000 итераций.
* **Двухфакторная энтропия:** Для хэширования используется уникальная соль (Salt) для записи и глобальный перец (Pepper) из переменных окружения.
* **Защита от брутфорса и тайминг-атак:** Безопасное сравнение строк через `hmac.compare_digest` и искусственная задержка при неверном вводе.
* **Безопасность буфера:** Автоматическая очистка буфера обмена через заданный таймаут.
* **Параметризованные SQL-запросы:** Полная защита от SQL-инъекций (`WHERE service = ?`).
